# assets

Recursos estáticos del sistema.

## Contenido actual

- `css/`: hojas de estilo globales (`styles.css`) y de autenticación (`auth.css`).
- `icons/`: carpeta reservada para iconos propios del proyecto.
- `imgs/`: carpeta reservada para imágenes del proyecto.

## Nota

Actualmente los iconos de interfaz se cargan principalmente desde Bootstrap Icons (CDN) en los layouts.
