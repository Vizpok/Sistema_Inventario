# sql

Scripts SQL del sistema. Idealmente mantener un solo script base versionado aquí.

## Estándar recomendado

- Mantener un script base único, por ejemplo: `schema_inventario.sql`.
- Si hay datos de prueba, usar un archivo separado opcional: `seed_inventario.sql`.
- Pensado para ejecución local en MySQL/MariaDB con XAMPP.
